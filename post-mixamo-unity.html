Hướng Dẫn: Tải Model & Animation Từ Mixamo Cho Unity (Chuẩn 2025)

Đăng bởi Trần Hữu Đạt | Ngày 14 tháng 10, 2025
Unity Game Dev 3D

Khi làm game, việc tìm kiếm model nhân vật 3D có sẵn animation là một trong những bước đầu tiên. Mixamo của Adobe là một công cụ tuyệt vời và miễn phí cho việc này. Tuy nhiên, để tải tài nguyên đúng cách và tối ưu cho Unity, bạn cần tuân theo một quy trình chuẩn. Dưới đây là hướng dẫn chi tiết nhất.

Giao diện chính của Adobe Mixamo.

Quy trình tải đúng cách

Bạn sẽ lặp lại thao tác Tìm kiếm → Chọn → Tải về cho mỗi animation bạn cần. Đây là cách làm linh hoạt và giúp quản lý project gọn gàng.

Ví dụ, để có 2 trạng thái cơ bản là Idle (đứng yên) và Walking (đi bộ), bạn cần tải ít nhất 3 file riêng biệt.

1. File 1: Nhân vật ở tư thế T-Pose (Quan trọng nhất)

Đây là file chứa model 3D hoàn chỉnh (lưới, xương, texture).

  • Tìm animation với từ khóa: "T-Pose".
  • Nhấn nút Download.
  • Trong cửa sổ thiết lập, chọn Skin: With Skin. Đây là bước quyết định để tải cả ngoại hình nhân vật.
  • Lưu file với tên gợi nhớ, ví dụ: MyCharacter.fbx.
Chọn 'With Skin' khi tải model nhân vật gốc.

2. File 2: Animation Đứng yên (Idle)

File này sẽ chỉ chứa dữ liệu chuyển động, không cần model đi kèm.

  • Tìm animation với từ khóa: "Idle" và chọn một kiểu bạn thích.
  • Nhấn nút Download.
  • Trong cửa sổ thiết lập, chọn Skin: Without Skin. Việc này giúp file cực nhẹ.
  • Lưu file với tên: Idle.fbx.

3. File 3: Animation Đi bộ (Walking)

Tương tự như file Idle, file này cũng chỉ chứa dữ liệu animation.

  • Tìm animation với từ khóa: "Walking" và chọn kiểu phù hợp.
  • Nhấn nút Download.
  • Trong cửa sổ thiết lập, chọn Skin: Without Skin.
  • Lưu file với tên: Walking.fbx.

Kết quả

Sau khi tải xong, bạn sẽ có một thư mục chứa các file như sau, sẵn sàng để import vào Unity và bắt đầu tạo Animator Controller:

/YourAssetsFolder
├── MyCharacter.fbx
├── Idle.fbx
├── Walking.fbx
├── (Nếu cần) Running.fbx
└── (Nếu cần) Jump.fbx
Nhân vật đã sẵn sàng trong Unity để bắt đầu lập trình.

Đây là cách làm chuẩn và hiệu quả nhất khi làm việc với Mixamo và Unity, giúp bạn quản lý tài nguyên một cách chuyên nghiệp.
